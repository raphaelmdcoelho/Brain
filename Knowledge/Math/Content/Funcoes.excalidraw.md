Coeficientes ^4TjQq2uM

a > 0 -> Concavidade para cima
a < 0 -> Concavidade para baixo

b > 0 -> Tangente crescente (onde corta o eixo y)
b < 0 -> Tangente decrescente (onde corta o eixo y)

c > 0 -> A parábola corta o eixo y em valores acima de x
c < 0 -> A parábola corta o eixo y em valores abaixo de x
c = 0 -> A parábola corta o eixo y na origem ^KMbM6tRH
