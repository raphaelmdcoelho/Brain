
## Concept

It is important to keep project dependencies isolate from other host resources.

## Usage

### Creating Virtual Environments

```bash
python -m venv crew-ai-env
```

### Activating it

**On windows**

```bash
crew-ai-env\Scripts\activate
```

**On Linux**

```bash
source crew-ai-env/bin/activate
```

### Deactivate 

```bash
deactivate
```
